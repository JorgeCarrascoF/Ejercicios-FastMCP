# Ejemplos de consultas para el servidor MCP

Puedes pedirme las siguientes consultas para interactuar con el servidor MCP y sus herramientas:

- Obtener todos los productos registrados.
  - Ejemplo: "Obtén todos los productos"
- Consultar los detalles de un producto específico por su ID o nombre.
  - Ejemplo: "Mira los detalles del Laptop"
- Crear un nuevo producto especificando sus datos.
  - Ejemplo: "Crea un producto llamado Monitor con stock 20 y precio 200"
- Actualizar la información de un producto existente.
  - Ejemplo: "Actualiza el stock del Mouse a 60"
- Eliminar un producto por su ID o nombre.
  - Ejemplo: "Elimina el producto Teclado"
- Consultar los movimientos (entradas/salidas) de un producto.
  - Ejemplo: "Muestra los movimientos del producto Laptop"
- Registrar un movimiento de entrada o salida para un producto.
  - Ejemplo: "Registra una salida de 2 unidades del producto Mouse"
- Comprobar el stock disponible de un producto.
  - Ejemplo: "¿Hay suficiente stock del producto Laptop para 5 unidades?"
