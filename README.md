# Ejercicios FastMCP

Este repositorio contiene una colección de ejercicios prácticos relacionados con la integración y uso de Model Context Protocol (MCP), OpenAPI y Gemini, organizados en diferentes carpetas según la temática y el tipo de ejercicio.

## Estructura del repositorio

- **Ejemplo1 -- Servidor y cliente MCP/**
  - Ejercicios que muestran cómo implementar y consumir un servidor MCP, incluyendo ejemplos de acceso desde Copilot y documentación relevante.
  - Subcarpetas:
    - `Acceso desde Copilot/`: Ejemplos y consultas sobre cómo interactuar con MCP usando Copilot.
    - `Servidor MCP/`: Código fuente de un servidor MCP y cliente, junto con instrucciones para ejecutar en Docker.

- **Ejemplo2 -- OpenAPI/**
  - Ejercicios enfocados en la creación y consumo de APIs RESTful utilizando OpenAPI y Spring Boot.
  - Incluye ejemplos de inventario, archivos de configuración, pruebas y documentación para desplegar servicios con Docker Compose.

- **Ejemplo3 -- Gemini/**
  - Ejercicios relacionados con el análisis de código y el uso de Gemini para tareas de inteligencia artificial y procesamiento de datos.
  - Incluye scripts de Python, archivos de datos y notebooks para experimentación.

## Temática

Los ejercicios están diseñados para explorar diferentes tecnologías y metodologías de integración entre sistemas, incluyendo:

- **Model Context Protocol (MCP):** Protocolo para la comunicación entre modelos y aplicaciones, facilitando la interoperabilidad y el acceso a servicios de IA.
- **OpenAPI:** Especificación para la definición de APIs RESTful, permitiendo la generación automática de documentación y clientes.
- **Gemini:** Ejercicios de análisis de código y datos, aplicando técnicas de IA y machine learning.

Cada carpeta contiene ejemplos prácticos, documentación y scripts que permiten aprender y experimentar con las tecnologías mencionadas. Se recomienda revisar los archivos `README.md` de cada subcarpeta para instrucciones específicas de ejecución y configuración.
